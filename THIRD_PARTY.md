# Third-Party Software

This repository builds and packages third-party open-source software.
The following components are included but remain licensed under their
original upstream licenses:

- [QEMU](https://gitlab.com/qemu-project)
- [libvirt](https://gitlab.com/libvirt/libvirt)
- [Seabios](https://github.com/coreboot/seabios)
- [OVMF](https://github.com/tianocore/edk2)
- [libtpms](https://github.com/stefanberger/libtpms)
- [swtpm](https://github.com/stefanberger/swtpm)
